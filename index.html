知识库

沉淀知识技能，打造技术体系

稳定性知识库

稳定性是一个软件的基础保证，同时它也是一项目需要持续建设的软件工程

软件架构知识库

如果数据结算和算法是一个软件是血和肉，那么软件架构就是一个软件的骨架，好的骨架，可以让你的软件跑的更稳更快

中间件知识库

中间件位于各类应用/服务与操作系统/数据库系统以及其他系统软件之间，主要解决分布式环境下数据传输、访问、系统构建等问题

大数据知识库

指来自不同源头的结构化和非结构化数据，并且普通的数据处理软件已不足以处理的大而复杂的数据集

开源学习

关注开源，提升技能，造轮子，也是一种创造

Spring Boot

由Pivotal团队提供的全新框架，其设计目的是用来简化新Spring应用的初始搭建以及开发过程

SOFAStack

一套用于快速构建金融级云原生架构的中间件，也是在金融场景里锤炼出来的最佳实践

Apache Pulsar

一个分布式的消息发布/订阅传递平台，具有非常灵活地消息模型和一个直观的客户端API

Apache Doris

百度开源的简单易用、高性能和统一的分析数据库

Polaris Mesh

一个支持多语言、多框架的云原生服务发现和治理中心

Apache Flink

一个集合众多具有竞争力特性于一身的第三代流处理引擎
